The simplest way to add Google tags (GA4, Ads) to your WordPress site. No bloat, no complexity - just tags.

== Description ==

Super Simple GTags does one thing really well - it adds Google tracking tags to your WordPress site with zero fuss.

Features:
* Add multiple GA4 and Google Ads tags
* Debug mode for developers
* Zero bloat - just the features you need
* Simple interface
* Loads tags efficiently in your header

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/super-simple-gtags`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > GTags Settings
4. Add your measurement ID(s)
5. Save changes

== Frequently Asked Questions ==

= Where do I find my measurement ID? =

GA4: Your measurement ID starts with "G-" and can be found in your Google Analytics Admin settings.
Google Ads: Your conversion ID starts with "AW-" and can be found in your Google Ads account.

= Can I add multiple tags? =

Yes! Click "Add Another Tag" to add as many GA4 or Google Ads tags as you need.

= What's Debug Mode? =

Debug mode enables Google's debug features for developers. Only enable this if you need to troubleshoot your tracking implementation.
